Proyecto: CyberCirujaZO.
Descripción: Este es el proyecto del sitio web del Nodo CyberCiruja Zona Oeste.
El mismo toma como base a la página principal de CyberCirujas y mejora su diseño.
Para esto será desarrollado con lo aprendido a lo largo del Curso de Front End JS.
Al finalizarse el proyecto, será entregado como regalo al Nodo Zona Oeste de CyberCirujas.
